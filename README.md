# PWA Zdrowie

Nowoczesna, bezpieczna aplikacja progresywna do zarządzania zdrowiem, lekami i wizytami lekarskimi.
Zintegrowana z Firebase Firestore.

## Jak zacząć lokalnie

1. Zainstaluj zależności:
   `npm install`
2. Uzupełnij plik `.env` swoimi kluczami Firebase (instrukcja w pliku `instrukcja.md`).
3. Uruchom aplikację:
   `npm run dev`

Szczegółowa instrukcja wdrożenia znajduje się w pliku `instrukcja.md`.
